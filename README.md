# RedBus Homepage Clone

A static front-end clone of the RedBus homepage built using only HTML and CSS.

This project recreates the visual design and layout of the RedBus website while focusing on clean structure, responsive styling, and modern CSS practices. The website is completely static and does not use JavaScript, animations, transitions, or transformations.

## Preview

This project includes:

- Navigation Bar
- Hero Banner Section
- Bus Search Interface
- Train Booking Section
- Offers Section
- Government Bus Booking Cards
- Testimonials Section
- App Download Banner
- SEO Information Section
- Footer Content

## Technologies Used

- HTML5
- CSS3

## Features

- Pure HTML and CSS implementation
- Static website design
- Responsive layout structure
- Clean and organized code
- No JavaScript used
- No animations
- No transitions
- No transforms
- Lightweight and beginner-friendly project
